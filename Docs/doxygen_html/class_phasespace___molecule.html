Detailed Description

This is the base molecule class. There will be several derivatives of this class that build on it's functionality for specific subtypes of Phasespase Theory (PST) calculations. This class stores various data relating to a molecule or fragment. It also has a number of functions that calculate values related to the fragment, which are used by the core PST state count class (and elsewhere, such as in the run functions of some calculation class derivatives). This class knows how to parse an input file and populate all of the data in the files relating to itself. This class also has some functions that write output specific to itself to a file.
